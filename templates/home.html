Description




Website

This website allows the user to draw a digit from 0 to 9 included. After submitting its drawing, the user will receive a prediction of the drawn number.

User Interface

In the drawing window, the user is asked to draw a digit from 0 to 9. The user can clear the window or submit the drawing. After a couple of seconds, the prediction model will try to retrieve the number drawn by the user.

Prediction Model

To make the prediction, we are using a real-time digit classifier: a convolutional neural network (CNN). This deep-learning model is one of the most powerful and accurate network for image recognitions and image classifications.

Performance

The model was trained and evaluated with the famous MNIST dataset: description here
The performance of the model on the test set is 99.33%.
